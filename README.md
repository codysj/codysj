#### 📌 Selected projects

**[AI-Backtest-Lab](https://github.com/codysj/AI-Backtest-Lab)** &nbsp;·&nbsp; *Python · FastAPI · Next.js · LangGraph*
Event-driven backtesting engine with a natural-language strategy builder and a LangGraph research copilot. The AI layer drafts strategies into a constrained DSL — no executable code generation, explicit approval gates before any run.

**[Vantage](https://github.com/codysj/Vantage)** &nbsp;·&nbsp; *Python · FastAPI · React · HuggingFace*
Prediction market intelligence dashboard. Correlates Polymarket price movement, news sentiment, and whale activity on one timeline to surface lead/lag relationships.

**[College-Explorer](https://github.com/codysj/College-Explorer)** &nbsp;·&nbsp; *Python · FastAPI · Next.js · Postgres*
Decision-support platform for prospective students. Deterministic ranking with normalized weights, hard constraints, and reason-code explanations — no black-box scoring.
